# Project Conventions

## UI Header Button Design
To maintain consistent, high-end premium aesthetics across the application, buttons placed on top of the blue header sections must follow this exact styling standard:

- **Style**: Translucent glassmorphism styled button overlay.
- **Background**: `bg-white/18` (semi-transparent white backdrop).
- **Border**: `border border-white/22` (very thin, soft translucent border).
- **Rounded Corners**: `rounded-[24px]` (high curvature capsule pill or premium rounded layout).
- **Typography & Layout**: Bold white text with horizontal flex alignment grouping icons side-by-side with text labels (no stacked or wrapped layouts).
- **Interactive States**: Smooth scale transitions on hover and active click states (e.g. `hover:bg-white/22 active:scale-[0.98]`).
